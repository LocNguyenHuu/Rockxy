# Changelog

<Update label="March 2026" description="v0.1.0">
  ### Added
  - SwiftNIO-based proxy engine for HTTP/HTTPS traffic capture
  - HTTPS interception via per-host certificate generation (swift-certificates)
  - SwiftUI + AppKit hybrid 3-column layout (sidebar, request list, inspector)
  - Certificate management with Keychain integration
  - Rule engine: breakpoint, map local/remote, block, throttle, modify header
  - Log capture engine (OSLog streams, process stdout/stderr)
  - Analytics engine: error grouping, performance percentiles, trend tracking
  - SQLite session persistence with 50,000-entry in-memory ring buffer
  - HAR export and cURL copy
  - WebSocket frame inspection
  - GraphQL operation detection
  - Request replay via URLSession
</Update>
