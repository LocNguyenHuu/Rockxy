# Insomnia

Insomnia is supported through an explicit manual proxy and certificate path in Developer Setup Hub.

## Support in Rockxy

- Manual setup available
- No automation entry point today
- Validation-backed setup

## What Rockxy currently provides

- Insomnia proxy preference guidance
- certificate trust guidance for the exported Rockxy PEM
- a cURL preflight path before app-level setup
