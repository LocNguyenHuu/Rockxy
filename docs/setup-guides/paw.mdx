# Paw

Paw follows a slightly different feel from Postman or Insomnia because it aligns more closely with the macOS system-proxy path.

## Support in Rockxy

- Manual setup available
- No automation entry point today
- Validation-backed setup

## What Rockxy currently provides

- a system-proxy-oriented setup path
- certificate guidance for trusting the exported PEM in the login keychain
- a cURL preflight before Paw traffic is tested
