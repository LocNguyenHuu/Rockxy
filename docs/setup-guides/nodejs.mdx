# Node.js

Node.js is a first-class Developer Setup Hub runtime target.

## Support in Rockxy

- Manual setup available
- Automatic Setup available
- Validation-backed setup

## What Rockxy currently provides

- snippets for `axios`, built-in `https`, and `got`
- proxy and certificate guidance from the live Rockxy runtime
- a Dev Hub validation path before you test real app traffic

## What you still need to do

- configure the Node.js client to use Rockxy as its proxy
- trust the exported root certificate for HTTPS interception
- confirm one visible request before moving to the full app flow
