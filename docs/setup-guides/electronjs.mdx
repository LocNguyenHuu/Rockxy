# ElectronJS

ElectronJS is documented separately because desktop-app traffic often sits outside browser DevTools and may need explicit proxy configuration.

## Support in Rockxy

- Manual setup available
- No automation entry point today
- Validation-backed setup

## What Rockxy currently provides

- a `--proxy-server` launch path
- a `session.setProxy` main-process path
- guidance for trusting the Rockxy certificate with `NODE_EXTRA_CA_CERTS`
