# Vision Pro

Vision Pro currently follows the iOS device class of setup in Rockxy.

## Support in Rockxy

- Manual setup available
- No automation entry point today
- Certificate sharing support available

## What Rockxy currently provides

- an iOS-class manual setup path
- guidance for network reachability and certificate trust
- Dev Hub placement as a real target instead of leaving it undocumented
