# cURL

cURL is one of the fastest ways to validate whether Rockxy is capturing traffic correctly.

## Support in Rockxy

- Manual setup available
- Automatic Setup available
- Validation-backed setup

## What Rockxy currently provides

- direct `--proxy` examples
- environment-variable examples using `HTTP_PROXY` and `HTTPS_PROXY`
- certificate guidance for HTTPS interception

## Why this target matters

cURL is often the best first probe before you debug a more complex browser, framework, or device flow.
