# Docker

Docker is supported as an environment target in Developer Setup Hub because container traffic needs explicit proxy and certificate plumbing.

## Support in Rockxy

- Manual setup available
- No automation entry point today
- Validation-backed setup

## What Rockxy currently provides

- a `docker run` validation pattern
- guidance for using `host.docker.internal`
- guidance for mounting the Rockxy PEM into the container path
