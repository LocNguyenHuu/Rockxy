# tvOS and watchOS

tvOS and watchOS follow the same broad setup class as iOS devices and simulators.

## Support in Rockxy

- Manual setup available
- No automation entry point today
- Certificate sharing support available

## What Rockxy currently provides

- guidance that maps these targets back to the iOS device and simulator setup classes
- certificate sharing support where the runtime path needs it
