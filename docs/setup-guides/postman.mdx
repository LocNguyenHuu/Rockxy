# Postman

Postman is a strong fit for Rockxy because it already centers around explicit request sending and repeatable API calls.

## Support in Rockxy

- Manual setup available
- No automation entry point today
- Validation-backed setup

## What Rockxy currently provides

- a Postman proxy settings path
- certificate guidance for the Rockxy PEM
- a cURL preflight before you change the app setup
