# Android Emulator

Android Emulator uses a manual proxy and user CA setup path, but iteration is usually faster than on a physical device.

## Support in Rockxy

- Manual setup available
- No automation entry point today
- Certificate sharing support available

## What Rockxy currently provides

- emulator-specific guidance using `10.0.2.2`
- certificate sharing support from the Mac
- app-level trust guidance for debug Android builds
